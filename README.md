# 📝 Todo App

A simple full-stack Todo application to create, update, and delete tasks.

---

## 🚀 Features

- Add a new task  
- Edit a task  
- Delete a task  
- Mark task as completed  

---

## 🛠️ Tech Stack

- Frontend: React  
- Backend: Node.js, Express  
- Database: MongoDB  

---

## 📁 Project Structure

```
todo_app/
├── client/   # frontend
├── server/   # backend
└── README.md
```

---

## ⚙️ Setup

### Clone the repo

```bash
git clone https://github.com/hemanth2k6/todo_app.git
cd todo_app
```

### Backend

```bash
cd server
npm install
npm start
```

Create `.env` file:

```
MONGO_URI=your_mongodb_uri
PORT=5000
```

### Frontend

```bash
cd client
npm install
npm start
```

---

## 🌐 Usage

Open `http://localhost:3000` and use the app.
